This project showcases an interactive Excel Dashboard developed using a banking dataset inspired by IDBI Bank operations. The main objective of this project is to transform raw banking data into meaningful insights through data analysis and visualization techniques in Microsoft Excel.

The dashboard is designed to help users quickly understand key performance indicators (KPIs), customer distribution, account performance, and loan-related trends. It demonstrates how Excel can be used as a powerful tool for business intelligence and decision-making.

🎯 Project Objective

The goal of this project is to:

Analyze banking data efficiently
Present insights in a visually appealing format
Enable interactive filtering for better exploration
Demonstrate Excel dashboarding skills for real-world applications

This project is particularly useful for beginners and intermediate users who want to build strong foundations in data analysis and dashboard creation using Excel.

Total Customers

Total Loan Amount
